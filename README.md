# Mini web prodavnica 
 
## Sportska prodavnica
 
## Tema prodavnice 
Sportska oprema, lopte, garderoba
 
## Opis projekta 
Ova web aplikacija predstavlja modernu sportsku prodavnicu razvijenu u WordPress-u uz korišćenje WooCommerce platforme. Omogućava pregled i kupovinu sportske opreme, odeće i obuće kroz intuitivan i responzivan korisnički interfejs.
Glavne funkcionalnosti uključuju prikaz proizvoda po kategorijama, pretragu proizvoda, korpu za kupovinu, upravljanje porudžbinama i sigurno online plaćanje. Aplikacija je optimizovana za rad na desktop i mobilnim uređajima, pružajući korisnicima jednostavno i efikasno iskustvo kupovine.
